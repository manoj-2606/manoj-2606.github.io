## Setup on GitHub Pages

1. Create a new GitHub repo named exactly: `manoj-2606.github.io`
2. Upload all files keeping the folder structure:
   ```
   index.html
   assets/css/style.css
   assets/js/main.js
   assets/images/profile.jpg
   resume/manoj_resume.pdf
   README.md
   ```
3. Go to repo Settings → Pages → Source: Deploy from branch → Branch: `main` → `/root`
4. Save. Site goes live at `https://manoj-2606.github.io` within 2 minutes.
